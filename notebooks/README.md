# Notebooks

This directory contains Jupyter notebooks for data exploration, analysis, and visualization of building energy benchmarking results.

## Recommended Notebooks

- **01_data_exploration.ipynb**: Explore the sample building energy data
- **02_benchmarking_analysis.ipynb**: Analyze benchmarking results and performance metrics
- **03_visualization.ipynb**: Create visualizations and dashboards for energy performance

## Getting Started

1. Install Jupyter: `pip install jupyter`
2. Launch Jupyter: `jupyter notebook`
3. Create and save your notebooks in this directory
